This file implements an MCP (Model Context Protocol) server called "lucas-never-quits-mcp" using FastMCP that provides document management capabilities. It exposes two tools for reading and writing files (`read_doc` and `write_file`), and two resources for listing and fetching markdown documents from the `./docs` directory. The server runs with stdio transport and requires Python 3.9+ with the `mcp[cli]>=1.0.0` dependency.
